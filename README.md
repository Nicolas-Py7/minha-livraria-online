# 📚 Livraria Online

Uma aplicação web de e-commerce para venda de livros, desenvolvida com foco em usabilidade, desempenho e experiência do usuário.

---

## 🚀 Funcionalidades

* 🔎 Busca de livros por título, autor ou categoria
* 📖 Página de detalhes do livro
* 🛒 Carrinho de compras
* 💳 Finalização de pedido (checkout)
* 👤 Cadastro e login de usuários
* 📦 Histórico de pedidos
* ⭐ Avaliações e comentários de livros

---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** (ex: React, Vue, HTML, CSS, JavaScript)
* **Backend:** (ex: Node.js, Express, Django)
* **Banco de Dados:** (ex: MongoDB, PostgreSQL, MySQL)
* **Autenticação:** (ex: JWT, OAuth)
* **Outros:** (ex: Docker, Stripe API, Firebase)

## ⚙️ Como Executar o Projeto

### Pré-requisitos

* Node.js instalado
* Gerenciador de pacotes (npm ou yarn)
* Banco de dados configurado

### Passos

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/livraria-online.git
   ```

2. Acesse o diretório:

   ```bash
   cd livraria-online
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Configure as variáveis de ambiente:

   ```
   .env
   ```

5. Execute o projeto:

   ```bash
   npm run dev
   ```

---

## 🔐 Variáveis de Ambiente

Exemplo de configuração:

```

DB_USER=root
DB_PASS=senha
```
